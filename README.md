# NWVG Tijdrit

Blazor WebAssembly applicatie voor het registreren en bijhouden van tijdritten voor wielervereniging NWVG.

## Functies

- 👥 **Deelnemers beheren** - Toevoegen, wijzigen en verwijderen van renners
- 🏁 **Tijdritten** - Aanmaken en beheren van tijdritten met rondetijden
- 🏆 **Uitslagen** - Klassementen per categorie bekijken
- 📊 **Historie** - Progressie per deelnemer met grafieken
- 💾 **Backup** - Export/import van data als JSON

## Technologie

- .NET 8 Blazor WebAssembly (standalone)
- Browser Local Storage voor data opslag
- Bootstrap 5 + Bootstrap Icons
