# Logo bestanden

Plaats hier de volgende bestanden:

1. **logo-nwvg.png** - Het hoofdlogo (het bestand dat je hebt geüpload)
2. **favicon.ico** of **favicon.png** - Kleine versie voor browser tabblad (32x32 of 16x16 pixels)

## Hoe favicon maken:
1. Ga naar https://favicon.io/favicon-converter/
2. Upload je logo
3. Download de gegenereerde bestanden
4. Kopieer `favicon.ico` naar `wwwroot/`
